♠ Blackjack -- Java Swing (Las Vegas Rules)
==========================================

A fully featured **Blackjack game** built in Java using **Swing**, designed to replicate an authentic Las Vegas casino experience.\
Includes advanced gameplay mechanics, smooth UI rendering, and detailed player statistics.

* * * * *

🎮 Features
-----------

-   🃏 **6-Deck Shoe**
-   💰 **Chip-based betting system**
-   ✂️ **Split up to 4 hands**
-   ✌️ **Double Down**
-   🛡️ **Insurance (Dealer Ace)**
-   🏳 **Surrender (early)**
-   🎯 **Blackjack pays 3:2**
-   👨‍💼 Dealer stands on soft 17
-   📊 Live stats tracking:
    -   Wins / Losses / Pushes
    -   Blackjacks
    -   Balance
-   📋 Hand history tracker
-   🎨 Custom-rendered UI (no images, all Java Graphics)

* * * * *


⚙️ Requirements
---------------

-   Java JDK **8 or higher**
-   Any IDE (recommended: IntelliJ or VS Code)

* * * * *

🚀 How to Run
-------------

### Option 1: Run from Source

javac Blackjack.java\
java Blackjack

* * * * *

### Option 2: Run with JAR (if included)

java -jar Blackjack.jar

* * * * *

🧠 Game Rules
-------------

-   Blackjack pays **3:2**
-   Dealer stands on **Soft 17**
-   You can:
    -   Hit
    -   Stand
    -   Double on any 2 cards
    -   Split pairs (up to 4 hands)
    -   Surrender on first move
-   Insurance available when dealer shows an Ace

* * * * *
