# Zing Zing Zingbah Racing
**Glowstar Cup Edition**

A top-down Boohbah racing game built with Java Swing.

## Cup format

- Three circuits: `Sky Ribbon`, `Moon Garden`, and `Sunset Harbor`
- Five racers compete across the full cup
- Points are awarded by finishing position each race
- The highest total after the last circuit wins the cup

## Controls

| Key | Action |
|-----|--------|
| `Up` | Accelerate |
| `Down` | Brake / reverse |
| `Left / Right` | Steer |
| `Space` | Use your move |
| `Enter` | Advance to the next circuit after results |
| `Esc` | Leave the current cup |

## Run
**Download the file**  
**Open it in an IDE**
**Locate game.java, right click it and press run**
**OR in the terminal**
```bash
javac *.java
java Game
```
